# Class Crash

A small Next.js app that helps UC Berkeley students quickly discover interesting lectures to sit in on during their free time.

## Tech stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Getting started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000` in your browser.

All course data lives in `data/courses.json` and is loaded on the client; there is no backend.
